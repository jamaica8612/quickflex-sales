보안운영 데이터는 계정 기준으로 저장합니다

퀵플은 운영 데이터를 Supabase 계정 기준으로 저장하고, 외부 OCR·API 처리는 브라우저에서 키를 직접 드러내지 않는 방식으로 다룹니다. 소개 페이지에서는 복잡한 설정 대신, 사용자가 안심하고 기록할 수 있다는 점만 짧게 안내합니다.

  • 가입 요청 후 관리자 승인된 계정만 앱 사용
  • 운영 데이터는 로그인한 계정 기준으로 분리 저장
  • API 키를 브라우저 화면에 직접 노출하지 않는 흐름 유지
