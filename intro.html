2기록 한 번이면 정산 자동

오늘 돈 구역과 배송건수만 입력하면 단가 곱셈 정산이 자동으로 계산됩니다. 같은 prefix(예: 322A·B·C) 라우트는 한 줄로 묶여 빠르게 입력됩니다.

  • 구역 × 단가 × 건수 = 일 매출 자동
  • 이미 등록한 단가는 unit snapshot으로 보존 (단가 변경해도 과거 기록은 그대로)
  • 휴무 토글로 휴식일 한 번에 처리

3기간별 통계로 추세 파악

일·월·연·누적 4가지 보기로 실적 흐름을 확인합니다. 하루 평균이 떨어지면 바로 보이고, 월말 정산 예상액도 미리 가늠할 수 있습니다.

  • 일간/월간/연간/누적 탭
  • 정산기간 단위(매월 26일 ~ 익월 25일)로 자동 그룹
  • 일평균·근무일·휴무일 등 핵심 지표

4단가 관리와 정산표 OCR

구역별 기본 단가는 한 곳에서 관리합니다. 정산표 이미지는 OCR로 읽어 단가 후보를 자동 추출합니다.

  • 구역별 단일 기본 단가 (변경해도 과거 기록은 unit snapshot으로 안전)
  • 스케줄 이미지 → 일별 라우트 자동 등록
  • 정산표 이미지 → 단가 자동 갱신 (관리자)

5회원제와 관리자 화면

개인용 기록 앱에서 여러 기사가 함께 쓰는 앱으로 바뀌었습니다. 기사는 자기 기록만 관리하고, 관리자는 승인·차단·고정 Route·사용자별 매출을 한 화면에서 확인합니다.

  • 가입 요청 후 관리자 승인 방식
  • 일반 기사는 달력·통계·설정만 표시
  • 관리자는 요약·라우트·사용자·정산표 탭 사용

6사용 흐름은 간단하게

처음 쓰는 기사도 복잡한 설정 없이 시작할 수 있도록 로그인, 승인, 기록, 통계 확인 순서로 정리했습니다.

  • 관리자가 먼저 계정을 승인하면 앱 사용 가능
  • 고정기사는 담당 Route를 등록해 OCR 없이 기록 가능
  • 월말에는 통계와 정산표 OCR로 단가를 점검
